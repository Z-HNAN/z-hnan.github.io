==, ===

=== 值和类型完全一致，严格

== 值相等即可，其判别的依据如下

类型相同的情况下，会直接比较

类型不相同,会转换类型进行比较

  1. {}=={}, 两个对象进行比较，是比较堆内存的地址
  2. null==undefined 相等， / null !== undefined
  3. NaN != NaN
  4. [12] = ‘12’, 调用toString后，在进行字符串的比较
  5. 其余情况都是进行数字的比较
    • boolean true=1, false=0
    • number = number
    • string = Number(‘str’), 如果有非数字则会变为NaN
    • undefined = NaN
    • Object(object - Number(obj.toString()), null - NaN)
