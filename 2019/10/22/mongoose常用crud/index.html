理解mongoose的三个对象

应该有一个Schema->Model->Document的过程

Schema

约束对象，用来规范文档的结构的

Model

对应到MongoDB中的集合，也就是之前定义过的

Document

每一个collection中的文档对象，也就是操作的每个存储文档,也就是mongoose取出来的样子

const UserModel = mongoose.model('users', userSchema)
